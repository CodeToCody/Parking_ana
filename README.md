## 🚦 **陽明校區 停車管理原則**
1. **分區停車限制：**  
   - 📅 **平日 08:00 - 17:00**
   - 🏫 **教職員工 & 非住宿生：** 主要教學區與指定停車場  
   - 🏢 **住宿生：** 限停放 **所屬宿舍區**
   - 🚘 **共同停車區：** 環山停車場、活動中心北側、致和園區等

2. **非管制時間 (08:00 前、17:00 後、例假日)**  
   - 除「職務專用」及「身心障礙/婦幼專用」車格，其餘停車格不限區域  
   - **車輛須停放於標線內**

3. **凌晨 01:00 - 06:00 室內停車場加收隔夜費** (每日 100 元)
